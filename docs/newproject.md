# 해외직구 서비스 기획서

# 해외직구멀티샵 - 서비스 기획서

> AI 기반 리뷰 통합 플랫폼으로 해외직구의 정보 탐색 시간을 90% 단축
> 

**작성일:** 2025년 11월 26일

**버전:** 1.0 (MVP)

**개발 기간:** 4주

**타겟:** 2030 직장인 해외직구 이용자

---

## 1. 개요 (Executive Summary)

### 1.1 핵심 가치 제안 (Value Proposition)

**"전세계 인기 상품의 리뷰를 AI가 한눈에 요약해드립니다."**

해외직구멀티샵은 여러 해외 쇼핑몰에 흩어진 상품 정보와 리뷰를 한 곳에 모아, AI 기술로 요약하여 제공하는 큐레이션 플랫폼입니다. 바쁜 2030 직장인들이 점심시간이나 퇴근 후 짧은 시간 안에 신뢰할 수 있는 구매 결정을 내릴 수 있도록 돕습니다.

### 1.2 해결하려는 문제

**2030 직장인 해외직구 이용자의 주요 Pain Points:**

1. **시간 부족 문제**
    - 여러 해외 쇼핑몰(아마존, 알리익스프레스, iHerb 등)을 일일이 방문하여 상품을 비교하는 데 평균 30-60분 소요
    - 퇴근 후나 점심시간 등 제한된 시간에 효율적인 쇼핑 불가능
2. **언어 장벽**
    - 영어, 중국어, 일본어 등 외국어 리뷰를 이해하기 어려움
    - 번역기를 사용해도 맥락과 뉘앙스를 정확히 파악하기 힘듦
3. **정보 분산**
    - 같은 상품에 대한 리뷰가 여러 사이트에 흩어져 있어 종합적인 평가가 어려움
    - 리뷰 수가 많을수록 핵심 정보를 찾기 위해 스크롤하는 시간 증가
4. **신뢰성 검증 어려움**
    - 가짜 리뷰와 진짜 리뷰 구분 어려움
    - 어떤 리뷰를 신뢰해야 할지 판단 기준 부족

### 1.3 솔루션 개요

**핵심 기능 3가지:**

1. **AI 리뷰 인텔리전스**
    - Gemini/ChatGPT/Claude를 활용한 수백 개 리뷰의 자동 요약
    - 긍정 포인트와 부정 포인트를 명확하게 구분
    - 감성 분석을 통한 전체 평가 시각화
2. **다국어 리뷰 통합 & 번역**
    - 여러 쇼핑몰의 리뷰를 한 곳에서 확인
    - 실시간 한글 번역으로 언어 장벽 제거
    - 자체 한국어 리뷰 시스템으로 신뢰성 강화
3. **큐레이션 중심 쇼핑**
    - 검증된 인기 상품만 엄선하여 제공
    - 카테고리별 베스트 상품 추천
    - 원클릭으로 원본 쇼핑몰 이동 (구매 편의성)
    - 직접 구매 할 수 있도록 추 후 업데이트

### 1.4 비즈니스 모델

**수익 구조:**

- **Phase 1 (MVP):** 무료 서비스 제공, 시장 검증 집중
- **Phase 2 (3개월 후):**
    - 거래 수수료: 구매 전환 시 3-5% 커미션
    - 배송 대행 수수료: 통합 배송 서비스 제공 시 수수료
- **Phase 3 (6개월 후):**
    - 프리미엄 멤버십: 무제한 AI 요약, 독점 할인
    - 판매자 프로모션: 상품 노출 증대 서비스

**예상 시장 규모:**

- 국내 해외직구 시장: 연 6조원 (2024년 기준)
- 2030 직장인 타겟 시장: 약 2조원
- 목표 시장 점유율: 0.5% (100억원 GMV, 1년 차)

### 1.5 핵심 차별화 요소

| 경쟁사 | 우리 서비스 |
| --- | --- |
| **쿠팡 해외직구:** 단일 플랫폼, 제한된 상품 | **멀티 쇼핑몰 통합:** 전세계 모든 인기 상품 |
| **아마존 글로벌:** 영어 리뷰만, 한국 사용자 불편 | **AI 한글 요약:** 핵심만 빠르게 파악 |
| **알리익스프레스:** 신뢰성 낮은 리뷰 | **자체 리뷰 + AI 검증:** 신뢰도 높음 |

### 1.6 MVP 개발 목표 및 검증 가설

**4주 후 달성 목표:**

- ✅ 50개 이상의 인기 상품 등록
- ✅ 30명 이상의 베타 테스터 확보
- ✅ AI 리뷰 요약 100회 이상 생성
- ✅ 20개 이상의 자체 한국어 리뷰 작성
- ✅ 평균 세션 시간 3분 이상

**검증하려는 핵심 가설:**

1. "AI 리뷰 요약이 구매 결정 시간을 50% 이상 단축시킨다"
2. "사용자의 70% 이상이 AI 요약 기능을 유용하다고 평가한다"
3. "자체 한국어 리뷰가 신뢰도를 20% 이상 높인다"

### 1.7 MVP 이후 발전 방향

**단기 (1-3개월):**

- 자동 크롤링 시스템 구축
- 상품 수 500개 확대
- 판매자 직접 등록 기능

**중기 (3-6개월):**

- 직접 결제 및 배송 대행 서비스
- 모바일 앱 출시
- AI 개인화 추천 시스템

**장기 (6-12개월):**

- 해외 쇼핑몰과 공식 파트너십
- 라이브 커머스 연동
- 동남아시아 시장 진출

5. 원본 사이트로 이동하여 구매

## 3. 사용자 여정 맵 (User Journey Map)

### 시나리오 1: 신규 사용자의 첫 구매 여정 (김민지)

### Stage 1: 인지 (Awareness)

**상황:** 인스타그램 광고에서 "AI가 리뷰를 요약해준다"는 문구 발견

- **사용자 행동:** 광고 클릭 → 랜딩 페이지 방문
- **감정:** 호기심 😊
- **터치포인트:** 소셜 미디어 광고 → 홈페이지
- **Pain Point:** "또 하나의 쇼핑 앱? 기존 앱과 뭐가 다르지?"
- **해결:** 히어로 섹션에서 "3분 만에 100개 리뷰 요약" 명확한 가치 제시

### Stage 2: 탐색 (Exploration)

**상황:** 홈페이지에서 인기 상품 둘러보기

- **사용자 행동:**
    - 카테고리 "패션" 클릭
    - "나이키 에어맥스" 상품 클릭
- **감정:** 흥미로움 🤔
- **터치포인트:** 상품 목록 페이지 → 상품 상세 페이지
- **Pain Point:** "정말 AI 요약이 유용할까? 기존 리뷰랑 뭐가 다르지?"
- **해결:** 상품 상세 페이지 상단에 AI 요약 박스 강조 배치

### Stage 3: 고려 (Consideration)

**상황:** AI 리뷰 요약 확인 및 자체 리뷰 읽기

- **사용자 행동:**
    - AI 요약 섹션 확인: "편안한 착용감", "발볼 넓은 분들에게 추천" 등 핵심 정보 파악
    - 자체 한국어 리뷰 5개 스크롤
    - 다른 색상 옵션 확인
- **감정:** 신뢰 상승 😃
- **터치포인트:** AI 요약 박스, 자체 리뷰 탭, 이미지 갤러리
- **Pain Point:** "이 가격이 최저가인지 확인하고 싶어"
- **해결:** 여러 쇼핑몰 가격 비교 표시 (Phase 2 기능)

### Stage 4: 구매 (Purchase)

**상황:** 구매 결정 후 원본 사이트 이동

- **사용자 행동:**
    - "아마존에서 구매하기" 버튼 클릭
    - 아마존 사이트로 리디렉션
    - 아마존에서 직접 결제
- **감정:** 만족 😊
- **터치포인트:** CTA 버튼 → 외부 쇼핑몰
- **Pain Point:** "결제는 다른 사이트에서 해야 하네. 조금 불편해"
- **해결 (Phase 2):** 통합 결제 시스템 도입

### Stage 5: 사용 (Usage)

**상황:** 상품 수령 후 만족도 평가

- **사용자 행동:**
    - 2주 후 상품 수령
    - 실제 착용 후 만족
    - "내 리뷰 쓰기" 알림 받음
- **감정:** 만족 😍
- **터치포인트:** 이메일 알림 → 리뷰 작성 페이지
- **Pain Point:** "리뷰 쓰는 게 귀찮아"
- **해결:** 간단한 별점 + 한 줄 평으로도 작성 가능, 포인트 적립

### Stage 6: 재구매 (Retention)

**상황:** 다른 상품 구매를 위해 재방문

- **사용자 행동:**
    - 1주일 후 앱 재접속
    - "내가 본 상품"에서 비슷한 상품 추천 확인
    - "뷰티" 카테고리에서 새 상품 탐색
- **감정:** 충성도 높아짐 🥰
- **터치포인트:** 마이페이지, 추천 시스템, 위시리스트
- **Pain Point:** "원하는 상품이 품절됐어"
- **해결:** 재입고 알림 기능 (Phase 2)

### 시나리오 2: 재구매 사용자의 빠른 구매 (이준호)

### 빠른 여정 (5분 소요)

1. **점심시간** → 앱 접속 (1분)
2. **검색창**에 "무선마우스" 입력 (10초)
3. **AI 요약** 확인: "무소음 클릭", "배터리 6개월" (1분)
4. **가격 비교** 확인: 아마존 $25, 알리 $18 (30초)
5. **한국 리뷰** 3개 확인: "배송 2주", "가성비 최고" (2분)
6. **구매 버튼** 클릭 → 알리익스프레스 이동 (30초)

**핵심 인사이트:** 기존 30분 → AI 요약으로 5분 (83% 단축)

### 기능 2: AI 리뷰 통합 및 요약 시스템 ⭐

**설명:**

여러 해외 쇼핑몰의 리뷰를 수집하여 GPT-4/Claude로 자동 요약하고, 긍정/부정 포인트를 명확히 구분하여 제공.

**사용자 스토리:**

"2030 직장인으로서, 나는 100개가 넘는 리뷰를 일일이 읽지 않고 핵심만 빠르게 파악하고 싶다. 왜냐하면 점심시간이나 퇴근 후 제한된 시간에 구매 결정을 내려야 하기 때문이다."

**기술 구현 방법:**

- **Next.js API Route:** `/api/summarize-review`
    
    ```tsx
    import OpenAI from 'openai';
    
    export async function POST(req: Request) {
      const { reviews, productName } = await req.json();
    
      const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });
    
      const response = await openai.chat.completions.create({
        model: 'gpt-4',
        messages: [
          {
            role: 'system',
            content: `너는 해외 쇼핑 리뷰 전문가야. ${productName}에 대한 리뷰들을 분석하여:
            1. 전체 요약 (3-4문장)
            2. 긍정 포인트 (3-5개)
            3. 부정 포인트 (3-5개)
            4. 추천 대상
            형식으로 한글로 요약해줘.`
          },
          {
            role: 'user',
            content: `리뷰들: ${reviews.join('\n---\n')}`
          }
        ],
        temperature: 0.7,
        max_tokens: 800
      });
    
      return Response.json({ summary: response.choices[0].message.content });
    }
    
    ```
    
- **Supabase:**
    - AI 요약 결과를 `products` 테이블에 저장 (캐싱)
    - 24시간마다 업데이트 (Cron Job)
- **Client Component:**
    - "AI 요약 보기" 버튼 클릭 시 API 호출
    - 로딩 스피너 표시 (5초 소요)
    - 결과를 긍정/부정 섹션으로 시각화

**화면 구성:**

```
┌─────────────────────────────────────┐
│  🤖 AI 리뷰 요약                      │
├─────────────────────────────────────┤
│  📊 전체 평가: ★★★★☆ 4.3/5          │
│                                     │
│  ✅ 긍정 포인트                      │
│  • 착용감이 매우 편안함              │
│  • 디자인이 세련되고 다양함          │
│  • 쿠션감이 좋아 오래 걸어도 편함     │
│                                     │
│  ⚠️ 부정 포인트                     │
│  • 발볼이 좁은 편                   │
│  • 배송이 평균 2-3주 소요            │
│  • 가격대가 다소 높음                │
│                                     │
│  👤 추천 대상                        │
│  발볼이 좁고 스타일을 중시하는 분들   │
└─────────────────────────────────────┘

```

**예상 개발 시간:** 4일
